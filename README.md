# 💼 Portfólio Pessoal - HTML, CSS e JavaScript

## 📌 Descrição

Este projeto consiste em um **portfólio pessoal simples**, desenvolvido utilizando **HTML**, **CSS** e **JavaScript**, com o objetivo de praticar e consolidar conhecimentos nas tecnologias fundamentais do desenvolvimento web.

Além disso, o projeto demonstra uma forma prática de **expor trabalhos, habilidades e projetos**, servindo como base para a criação de um portfólio profissional.

---

## 🎯 Objetivo do Projeto

O principal objetivo deste projeto é:

- Praticar os fundamentos do desenvolvimento web
- Consolidar conhecimentos em HTML, CSS e JavaScript
- Aprender a estruturar uma página web responsiva
- Demonstrar como criar e apresentar um portfólio pessoal
- Servir como material de estudo e evolução contínua

Este projeto foi desenvolvido como parte do processo de aprendizado em desenvolvimento web.

---

## 🚀 Tecnologias Utilizadas

As seguintes tecnologias foram utilizadas no desenvolvimento do projeto:

- **HTML5** — Estrutura da página
- **CSS3** — Estilização e layout
- **JavaScript** — Interatividade e comportamento da aplicação

## ✨ Funcionalidades

- Exibição de informações pessoais
- Apresentação de projetos desenvolvidos
- Seção de habilidades
- Layout simples e organizado
- Navegação entre seções
- Estrutura responsiva (adaptável a diferentes telas)
